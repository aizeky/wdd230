Typography

Font #1 - Fjalla One

This font will be used for headers and other header text for the website. This font has a sans-serif element and also has a weight of boldness that will help emphasize attention to the text.

Font #2 - Cormorant Garamonde

This font will be used for the paragraph elements of the website. Cormorant Garamonde is a serif font and that detal will bring a formal element to the website to make it look professional.
